게임 소개

이 게임은 플레피버드처럼 왼쪽에서 오른쪽으로 진행하며 장애물을 피하는 2D 러너/플라잉 게임입니다. 배경과 타일맵 청크가 왼쪽으로 스크롤되며, 플레이어는 상승/하강 조작으로 벽 사이의 통로를 통과하고 아이템을 먹어 점수를 올립니다. 

추가로, 스테이지에는 색깔벽(Color Wall) 이 등장하며 플레이어는 필터 모드(색상)를 바꿔 특정 색깔벽을 “통과 가능한 상태(Trigger)”로 만들 수 있습니다. 

핵심 플레이

절차적(랜덤) 타일맵 청크 생성
일정 폭의 청크(Tilemap)를 계속 생성/삭제하면서 맵이 끊기지 않게 이어집니다. 청크마다 벽 2개와 각 벽의 통로(연속 3칸)가 랜덤으로 생성됩니다. 

난이도(속도) 점진 상승
시간이 지날수록 스크롤 속도가 커져 점점 어려워집니다(커브 기반). 배경 스크롤도 동일 컨셉으로 속도가 증가합니다. 

색깔벽 + 필터 모드 퍼즐
청크 중앙 근처에 색깔벽(빨/초/파/흰)이 생성될 수 있고, 플레이어가 A키로 필터 모드를 순환하면 현재 모드와 동일한 색깔벽은 트리거로 바뀌어 통과할 수 있게 됩니다. 

아이템/점수
청크마다 아이템이 고정 개수로 스폰되며(기본 6개), 먹으면 점수가 오르고 수집 애니메이션 후 제거됩니다. Best Score도 갱신됩니다. 


조작법

D 키: 상승(Fly) 

S 키: 급강하(Dive) 

A 키: 필터 모드(색상) 변경(White → Red → Green → Blue 순환) 


게임 시작 / 종료

시작 화면에서 Start 버튼을 누르면 게임이 시작되며, 시작 전에는 Time.timeScale = 0으로 정지 상태입니다. 

플레이어가 카메라 화면 밖으로 나가면 게임오버 처리되고 시간이 멈춥니다. 

GameOver 버튼으로 main 씬을 다시 로드합니다. 

나가기는 Alt+F4를 하셔야합니다


주요 스크립트 구성

PlayerController : 이동/상승/급강하 입력 및 충돌 시 전진 제어 

ChunkSpawner : 청크 생성/이동/삭제, 벽 패턴/색깔벽/아이템 스폰 

ColorFilter, ColorWallState, FilterMode : 필터 모드와 색깔벽 트리거 로직 

BG_Scroll : 배경 루프 스크롤 + 시간에 따른 속도 증가 

ScoreItem, UI_Fixer : 점수/베스트 표시, 시작 버튼, 게임오버 UI 처리 

Gameover : 화면 밖 판정 게임오버 + 재시작 

Followcam : 플레이어를 따라가는 카메라 로직
